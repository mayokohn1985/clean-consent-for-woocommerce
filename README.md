# Clean Consent for WooCommerce

Lightweight cookie consent plugin for WordPress and WooCommerce without bloat.

## Why this plugin?

Most cookie plugins are:
- slow
- bloated
- dependent on external services

Clean Consent is:
- lightweight
- developer-friendly
- performance-first

## Features

- No jQuery
- No external API calls
- Script auto-blocking (Google Analytics, Meta Pixel, etc.)
- Works with WooCommerce
- Fast and simple

## Roadmap

- Admin settings UI
- Script detection dashboard
- Category-based consent
- WooCommerce deep integration
